# KEY ANNOTATIONS:
#
# ✓ GPU0↔GPU1: NV18  — GOOD. Direct NVLink. Full 900 GB/s bidirectional.
# ✓ GPU0↔GPU7: NV18  — GOOD. Even cross-NUMA GPU pairs use NVLink, not PCIe.
# ✓ GPU0↔mlx5_0: PXB — GOOD. GPU0 and its NIC are on the same PCIe root complex.
#                        This is the GPUDirect RDMA path. Low latency.
# ✗ If GPU0↔GPU1 shows PHB or SYS instead of NV18:
#     → NVSwitch link down or NVLink disabled
#     → Check: nvidia-smi nvlink -s -i 0
#     → Check: dmesg | grep -i nvlink
#
# ✓ mlx5_0↔mlx5_1: PIX — GOOD. Two NICs sharing a PCIe switch. Expected.
# ✗ If any GPU↔GPU shows PIX or PHB:
#     → Run: nvidia-smi nvlink -e -i <gpu_id>  (error counters)
#     → If errors > 0: reseat GPU or contact NVIDIA support

# NUMA NOTE:
#   GPUs 0-3 + NICs 0-3: NUMA node 0 (CPU socket 0)
